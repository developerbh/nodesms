# nodesms
Node.js SMS Sender
===========================

A very simple basic Node.js SMS application using twilio service

Requirements
------------

* [Node.js](http://nodejs.org/)


Dependencies Needed
-----------------------

The only dependency needed is Express JS:  
`npm install twilio`
`npm install prompt`


Running
-------

Start the server by executing this command:  
`node SMS-Sender.js`
